所以先用Aspectj进行方法拦截，这地方涉及到了原生的Aspectj使用，网上资料比较少，大部分都是springAOP里面的

先是下载Aspectj必要的三个包

  • aspectjtools-1.9.4.jar 编译工具，用来编译切面到具体的class或者jar，
  • aspectjweaver-1.9.4.jar 注解支持，运行时代理， 编译时支持 @PointCut 等。
  • aspectjrt-1.9.4.jar 运行时支持，

写完之后将上述三个文件放置至同一包下
